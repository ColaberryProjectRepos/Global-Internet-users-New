Transforming the data in power Query Editor 9
Step 2: Transforming and cleaning the data in Power Query Editor :

  1. In this step, we will be transforming the data
  2. On the home page click on the transform data
  3. Start by renaming the first column that has a blank name to ID
  4. Rename Entitity to Country
  5. Rename Code to Country - Code
  6. Right-click on the Year and duplicate it then rename the duplicate to Date
  7. Right-click the Date column then change type from whole number to text. Then change type from Text to Date, If it asks you to replace current or add new step -make sure to Add New Step
  8. Click on the Close and Apply (Top Left)
