Step 12: Creating Yoy | Year Visualization Chart



  1. start by clicking on a Line and clustered column chart in the visualization pane
  2. Drag and drop the following values into the Chart and change the name if needed.
    1. X-axis
      1. calendar table
        1. Decades 
        2. Year 
    2. Column y-axis
      1. Measures/Dax table
      2. Average of No. of Internet Users = Users
    3. Line y-axis
      1. Users %Yoy
      2. Cellular %Yoy = Cell %Yoy
      3. Broadband %Yoy = Broad %Yoy
    4. Tooltips
      1. Cellular Subscription YTD = Cellular Subs
      2. Broadband Sub VS Cellular sub% = Broadband Subs
      3. Users %Yoy = "Internet users YoY%
  3. chart visual
    • Format visual
      1. X-axis
        1. callout value
          • font = Calibri
          • size = 9
          • color = #000000
        2. category label = OFF
      2. Y-axis
        1. callout value
          • font = Calibri
          • size = 9
          • color = #000000
        2. category label = OFF
      3. Secondary Y-axis = OFF
      4. Legend = ON
        • Position = top center
        • style = marker (circle default)
      5. Gridlines = OFF
      6. Lines 
        • Series  = All
        • Shape
          • line style = solid
          • Join type = round
          • Strock width = 5 px
      7. Colors 
        1. Broad %Yoy = #6B007B
        2. Cell %Yoy = #E66C37
        3. User %Yoy = #12239E
      8. markers = OFF
      9. data labels = OFF
      10. Series Labels = OFF
    • General
      1. Properties 
        • Size Height = 297
        • Size Width = 531
      2. Position
        • Horizontal = 218
        • vertical = 116
      3. Title = ON
        • title = Yoy | Year
        • heading =  heading 3
        • Font = Calibri
        • Size = 12
        • text color = #FFFFFF
        • background-color = #001E49
        • Horizontal alignment = center 
      4. Effects 
        • background = ON
          • color = #001E49
          • Transparency  = 77%
        • Visual border = ON
          • color = #B3B3B3
