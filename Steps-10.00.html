Step 3: Creating the Calendar table :
  1. In power bi on the ribbon we have Modeling Click on it then click on the new table button to create the new table.
  2. In the Dax remove the "table =" then Copy and paste the following code.
    1. Calendar = ADDCOLUMNS(CALENDAR(MIN('Global Internet Users'[Date]),MAX('Global Internet Users'[Date])),
"DateAsInteger", FORMAT ( [Date], "YYYYMMDD" ),

"Year", YEAR ( [Date] ),

"Monthnumber", FORMAT ( [Date], "MM" ),
"YearMonthnumber", FORMAT ( [Date], "YYYY/MM" ),

"YearMonthShort", FORMAT ( [Date], "YYYY/mmm" ),

"MonthNameShort", FORMAT ( [Date], "mmm" ),

"MonthNameLong", FORMAT ( [Date], "mmmm" ),

"DayOfWeekNumber", WEEKDAY ( [Date] ),

"DayOfWeek", FORMAT ( [Date], "dddd" ),

"DayOfWeekShort", FORMAT ( [Date], "ddd" ),

"Quarter", "Q" & FORMAT ( [Date], "Q" ),

"YearQuarter", FORMAT ( [Date], "YYYY" ) & "/Q" & FORMAT ( [Date], "Q" ),

"Decades", YEAR([Date]) - MOD(YEAR([Date]), -10))
  3. Now you have the Calendar Table
